# 🧮 Interactive Math Tables Project (C++)

This is my first interactive project built during my learning journey with **Programming Advices** (Dr. Mohamed Abu-Hadhoud) and **Git/GitHub** course (Mohamed Jaber).

### 🚀 Features:
- **Interactive Menu**: Choose between Addition, Subtraction, Multiplication, and Division.
- **Validation**: Ensures the user enters positive numbers only.
- **Formatting**: Beautifully aligned tables for all operations.
- **Styling**: Uses system commands for colors (Matrix/Hacker style) and sounds.

### 🛠 Tech Stack:
- **Language**: C++
- **Concepts**: Enums, Type Casting, Loops (do-while), Functions, and System Commands.
